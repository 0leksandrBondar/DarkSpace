# DarkSpace

This project is designed to learn how to work with the following technologies:
  -  Multithreading
  -  Qt QML
  -  Databases
  -  Network programming (TCP/IP)
  -  C++

# Tools
  - C++
  - Compiler MSVS 2019
  - QT Creator

# Libs
  - Qt6::Core
  - Qt6::Network
  - Qt6::Quick
